AR feature set

Unity provides powerful AR tools to create rich, deeply engaging augmented reality experiences that intelligently interact with the real world. Unity’s AR feature set provides a starting point for augmented reality development, spanning from head-mounted displays to mobile.

准备开始​

Unity’s AR feature set pre-installs the correct packages needed for AR development. Unity directly supports the following AR platforms:

  • ARCore
  • ARKit
  • Magic Leap
  • Hololens via OpenXR

For more information on how to start developing applications for these platforms in Unity, see the documentation in the list above.

To get started with AR Foundation and add AR features to your project, see:

  • AR Foundation documentation
  • AR Foundation Samples GitHub repository

配置​

To configure your Unity Project for AR, see Using AR Foundation.

Packages​

The AR feature set contains the following packages:

  • AR Foundation
  • ARKit
  • ARCore
  • Magic Leap
  • ARKit Face Tracking
  • OpenXR

示例​

The following samples are available for the AR feature set.

  • AR Foundation Samples GitHub repository

Resources​

To learn more about Unity’s full suite of AR platforms, watch What’s new in AR Foundation.

Related forums​

Reach out and connect with our developer community in Unity’s AR forum.

Public roadmap​

We’re on a mission to provide the best development platform for creating AR/VR experiences that distribute seamlessly to your target platforms. To understand our future trajectory, refer to the AR/VR product roadmap.
