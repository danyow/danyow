Supported rich text tags

The following table lists all the supported rich text tags:

标签描述示例
&lt;a&gt;Define a hyperlink. Use the href attribute to define the hyperlink URL.&lt;a href="https://www.unity.com"&gt`;Visit Unity!</a>`
&lt;align&gt;Change the text’s horizontal alignment. The supported values are:
  • left
  • center
  • right
  • justified
  • flush


If you put multiple alignment tags on the same line, the last one overrides the others. | &lt;align="left"&gt;Left-aligned&lt;/align&gt; | | &lt;allcaps&gt; | Convert text to uppercase. | &lt;allcaps&gt;Alice and Bob watched TV.&lt;/allcaps&gt; | | &lt;alpha&gt; | Change text opacity. It works with hexadecimal values. | &lt;alpha=#FF&gt;FF &lt;alpha=#CC&gt;CC &lt;alpha=#AA&gt;AA &lt;alpha=#88&gt;88 &lt;alpha=#66&gt;66 &lt;alpha=#44&gt;44 &lt;alpha=#22&gt;22 &lt;alpha=#00&gt;00 | | &lt;b&gt; | Render text in boldface. | The fox jumps over the &lt;b&gt;lazy dog&lt;/b&gt; | | &lt;br/&gt; | Forces a line break in text. | Break the line here &lt;br/&gt; New line starts | | &lt;color&gt; | Change text color or color and opacity. It supports color names and hexadecimal values. If you apply successive tags in the same text, the last one takes precedence over the others until you either add another tag or use a closing tag to end the current color’s scope. | &lt;color="red"&gt;Red &lt;color=#005500&gt;Dark Green &lt;#0000FF&gt;Blue &lt;color=#FF000088&gt;Semitransparent Red | | &lt;cspace&gt; | Change spacing between characters, either absolute or relative to the original font Asset. Use pixels or font units. Positive adjustments push the characters apart, negative adjustments pull them together. | &lt;cspace=1em&gt;Spacing&lt;/cspace&gt; is just as important as &lt;cspace=-0.5em&gt;timing. | | &lt;font&gt; | Change text font. | Would you like &lt;font="Impact SDF"&gt;a different font?&lt;/font&gt; | | &lt;font-weight&gt; | Change the text’s font weight to any of the weights defined in the font Asset. If you haven’t defined any font weights, you can still use 400 for normal, 700 for bold. | &lt;font-weight="100"&gt;Thin&lt;/font-weight&gt; | | &lt;gradient&gt; | Apply a color gradient to text. | &lt;gradient="Light to Dark Green - Vertical"&gt;gradient | | &lt;i&gt; | Render text in italics. | The &lt;i&gt;quick brown fox&lt;/i&gt; | | &lt;indent&gt; | Indent all text between the tag and the next hard line break. Use this tag to create text patterns, such as bullet points, that work with word-wrapping. Specify indentation in pixels, font units, or percentages. | &lt;indent=15%&gt;It is useful for things like bullet points.&lt;/indent&gt; | | &lt;line-height&gt; | Modify the line height relative to the default line height specified in the font Asset. Specify the line height in pixels, font units, or percentages. | &lt;line-height=50%&gt;Line height at 50% | | &lt;line-indent&gt; | Indent the first line after every hard line break. New lines created by word-wrapping are not indented. | &lt;line-indent=15%&gt;This line is indented. &lt;br/&gt;This line is also indented. | | &lt;lowercase&gt; | Convert text to lowercase. | &lt;lowercase&gt;Alice and Bob watched TV.&lt;/lowercase&gt; | | &lt;margin&gt; | Set the text horizontal margins. If you only want to adjust the left or right margin, you can use the &lt;margin-left&gt; or &lt;margin-right&gt; tag. Specify the margins in pixels, font units, and percentages. Negative values have no effect. | &lt;margin=5em&gt; | | &lt;mark&gt; | Highlight the text with a colored overlay. The overlay must be translucent (alpha less than 1) for the text to show through. | Text &lt;mark=#ffff00aa&gt;can be marked with&lt;/mark&gt; an overlay. | | &lt;mspace&gt; | Override a font’s character spacing and turn it into a monospace font. | Any font can become &lt;mspace=2.75em&gt;monospace, if you really want it. | | &lt;nobr&gt; | Keep a segment of text together. | You don't want &lt;nobr&gt;I M P O R T A N T&lt;/nobr&gt; things to be broken up. | | &lt;noparse&gt; | Prevent parsing of rich text tags. | Use &lt;noparse&gt;&lt;b&gt;&lt;/noparse&gt; for &lt;b&gt;bold&lt;/b&gt; text. | | &lt;pos&gt; | Set the horizontal caret position on the current line. Specify the horizontal position in pixels, font units, or percentages. | at &lt;pos=75%&gt;75% | | &lt;rotate&gt; | Rotates each character about its center. Specify the amount of rotation in degrees. Positive values rotate characters counter-clockwise. Negative values rotate them clockwise.
Rotation affects the spacing between characters, and might cause characters to overlap in some cases. Use the &lt;cspace&gt; tag to correct character spacing as needed. | Rotate text &lt;rotate="45"&gt;counter-clockwise&lt;/rotate&gt; | | &lt;s&gt; | Render a line across the text. | The &lt;s&gt;quick brown&lt;/s&gt; fox | | &lt;size&gt; | Adjusts the font size. Specify the new size in pixels, font units, or percentage. Pixel adjustments can be absolute (such as 5px) or relative (such as +1 or -1). Relative sizes are based on the original font size, so they’re not cumulative. | &lt;size=100%&gt;Echo &lt;size=80%&gt;Echo &lt;size=60%&gt;Echo &lt;size=40%&gt;Echo &lt;size=20%&gt;Echo | | &lt;smallcaps&gt; | Convert text to lowercase. | &lt;smallcaps&gt;Alice and Bob watched TV. | | &lt;space&gt; | Add a horizontal offset between itself and the rest of the text. Specify the offset in pixels or font units. | Give me some &lt;space=5em&gt; space | | &lt;sprite&gt; | Add a sprite from a sprite asset into your text. | &lt;sprite name="spriteName"&gt; | | &lt;strikethrough&gt; | Draws a line slightly above the baseline so it crosses out the text. | &lt;strikethrough&gt; Some text | | &lt;style&gt; | Apply a custom style to the text. | &lt;style="H1"&gt;Styles&lt;/style&gt; | | &lt;sub&gt; | Convert the text to subscript. | We have 1m&lt;sup&gt;3&lt;/sup&gt; of H&lt;sub&gt;2&lt;/sub&gt;O. | | &lt;sup&gt; | Convert the test to superscript. | We have 1m&lt;sup&gt;3&lt;/sup&gt; of H&lt;sub&gt;2&lt;/sub&gt;O. | | &lt;u&gt; | Underline the text. | &lt;u&gt;The lazy dog&lt;/u&gt; | | &lt;uppercase&gt; | Convert text to uppercase. | &lt;uppercase&gt;Alice and Bob watched TV.&lt;/uppercase&gt; | | &lt;voffset&gt; | Give the baseline a vertical offset. Specify the offset in pixels or font units. The offset is always relative to the original baseline. | Up &lt;voffset=1em&gt;up &lt;voffset=2em&gt;UP&lt;/voffset&gt; and &lt;voffset=-0.5em&gt;down&lt;/voffset&gt; we go again. | | &lt;width&gt; | Change the horizontal size of text area. | &lt;width=60%&gt;Those days are long gone&lt;/width&gt; | | |
