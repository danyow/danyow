购买收据

Unity IAP 以 JSON 哈希（其中包含以下键和值）的形式提供购买收据：

键值
Store正在使用的商店的名称，例如 GooglePlay 或 AppleAppStore
TransactionID该交易的唯一标识符，由应用商店提供
Payload因平台而异，详情如下。

iOS​

有效负载 (Payload) 因设备的 iOS 版本而异。

iOS 版本有效负载
iOS >= 7有效负载是以 base 64 编码的应用程序收据。
iOS < 7有效负载是 SKPaymentTransaction transactionReceipt。

Mac App Store​

有效负载是以 base 64 编码的应用程序收据。

Google Play​

有效负载是具有以下键和值的 JSON 哈希：

键值
jsonGoogle 提供的 JSON 编码字符串；INAPP_PURCHASE_DATA
signatureGoogle 提供的 json 参数签名；INAPP_DATA_SIGNATURE

通用 Windows 平台​

有效负载是由 Microsoft 指定的 XML 字符串
