ruby在mac上的升级

September 6, 2019 · One min read

Title: mac自带的ruby版本过低

安装rvm环境来安装ruby环境​

`curl -L https://get.rvm.io | bash -s stable`

查看当前版本​

`rvm list known`

安装指定ruby版本​

`rvm install 2.6.3`

done!
